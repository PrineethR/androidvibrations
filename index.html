Vibration API Demos

Test the Vibration API demos on a supported Android device.

by Prineeth

Demo 1 – Simple vibration

Vibrate for 500 ms using an array with a single value.

Demo 2 – Simple vibration (number)

Vibrate for 500 ms using a single numeric value.

Demo 3 – Fixed pattern

Vibrate for 500 ms six times, pausing 250 ms between each one.

Demo 4 – Stop vibrations

Stop any current vibration immediately.

Demo 5 – Custom pattern

Enter your own pattern as comma‑separated values or a JSON array, then play it. Example: 250, 100, 250, 100, 400 or [250, 100, 250, 100, 400].
